돌아가기

Python-Operator연산자

    산술 연산자
         -더하기 : x + y
         -빼기 : x - y
         -곱하기 : x * y
         -거듭제곱 : x ** y
         -나누기 : x - y
         -몫 : x // y
         -나머지 : x % y
      
    비교 연산자(관계연산자)
        ==  같으면 true, 틀리면 false
        !=  틀리면 true, 같으면 false
        <  a<b a가 b보다 작으면 true,크면 false
        >  a>b b가 a보다 작으면 true,크면 false
        <=  a<=b a가 b보다 작거나 같으면 true, 크면 false
        >=  a>=b a가 b보다 작거나 같으면 true, 크면 false
