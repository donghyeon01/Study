HTML-태그

  • <hn>태그
    • Heading   제목 태그로 h1~h6까지 존재하며, 숫자가 커질수록 크기가 작아진다.

  • <p>태그
    • Paragraph   문단 태그로 태그 앞 뒤로 한 줄씩 자동 띄어쓰기를 해준다.

  • <br>태그
    • Link break   줄바꿈 태그이다.

  • <pre>태그
    • <pre>태그 내부에서는 띄어쓰기나 Enter가 그대로 적용되어 글 작성시 추가적으로 <br>태그나 &nbsp;를 사용하지 않아도 된다.

  • <hr>태그
    • html 문서 내에 수평선을 삽입한다.

  • <li>태그
    • List item   목록 하나를 의미한다. 목록을 만들때마다 생성하여야 한다.

  • <ol>태그, <ul>태그
    • Ordered list   순서가 있는 목록을 만드는데 사용한다. <li>태그를 감싸면 순서에 따라 숫자가 붙여서 나온다
    Unordered list   순서가 없는 목록을 만드는데 사용한다. <li>태그를 감싸면 숫자나 기호를 붙이지 않고 나온다.

  • 글자 관련 태그
    • <B>태그는 글자를 굵게 표현합니다.
    <STRONG>태그는 글자를 굵게 표현합니다.
    <I>태그는글자를 기울여서 표현합니다.
    <STRIKE>태그는 글자에 취소선을 표시합니다.
    <U>태그는 글자에 밑줄을 표시합니다.
    <Sup>태그는 글자를 위 첨자로 표시합니다.
    <sub>태그는 글자를 아래첨자로 표시합니다.
    <TT>태그는 글자를 타자체로 표현합니다.
    <BIG>태그는 글자를 주변의 글자보다 크게표현합니다.
    <SMALL>태그는 글자를 주변의 글자보다 작게표현합니다.
